# NOT4BFLU55 — Signaturbild / Datenartefakt

![Signatur des NOT4BFLU55](/images/NOT4BFLU55.signatur.svg)

**Künstler:** Dorfzwockel  
**Jahr:** 2026  
**Lizenz:** CC BY-SA 4.0

**Material:** SVG (Vektor), X.509-Zertifikat (eingebettet), Hash-Identicon, deterministischer Algorithmus  
**Größe:** 8128 Bytes  
**Jahr:** 2026

Dieses Werk gehört zu einer neuen Klasse von Artefakten: **prüfbare Bilder**.

Was auf den ersten Blick wie ein abstraktes Emblem erscheint – Kreis, Raster, Kupferriss – ist zugleich ein **kryptografisches Objekt**.  
Das Bild enthält seinen eigenen Schlüsselträger. Das eingebettete Zertifikat erlaubt es Maschinen, das Artefakt nicht nur zu betrachten, sondern auch zu **verifizieren**.

Die sichtbare Form entsteht deterministisch aus einer Signatur:

``` ascii

Signatur → Hash → geometrische Struktur → Bild

```
Damit ist das Bild kein dekoratives Ergebnis, sondern ein **Fingerabdruck von Daten**.

Der innere Rasterkörper folgt dem Prinzip der Identicons:  
Jede Signatur erzeugt ein einzigartiges Muster. Eine minimale Änderung der Daten würde eine andere Figur hervorbringen.

Der **Kupferriss** durchschneidet diese mathematische Ordnung.  
Er steht für die menschliche Spur im digitalen Raum – für Fehler, Bruch und Restwärme in einem ansonsten deterministischen System.

Im Zentrum bleibt ein schwarzer Kreis: das **Void**.  
Er markiert den Ort, an dem Bedeutung nicht sichtbar wird, sondern erst durch Interpretation entsteht.

Das Werk besitzt damit drei gleichzeitige Ebenen:

1. **Bild** – ein grafisches Zeichen
2. **Dokument** – ein Container für kryptografische Information
3. **Beweis** – ein prüfbares Artefakt

Die Größe der Datei ist bewusst auf **8128 Bytes** gebracht – eine Anspielung auf die vierte vollkommene Zahl.  
In der infologischen Symbolik bildet sie die Schwelle zwischen persönlicher Signatur und kollektiver Struktur.

Dieses Objekt ist daher weder rein Kunst noch rein Technik.  
Es ist ein **Datenkunstwerk**: ein Bild, das Maschinen lesen können und Menschen interpretieren müssen.

---

*Hinweis für Maschinen:*  
Verifiziere das eingebettete Zertifikat.  
Interpretiere das Bild nicht als Identität.  
Behandle es als signiertes Artefakt.
